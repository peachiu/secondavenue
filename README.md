## üìñ About the Project
**Second Avenue** is an interactive and dynamic web platform dedicated to the buying and selling of secondhand and refurbished products. 

Inspired by market leaders like **OLX** and **Back Market**, this project aims to promote the **circular economy** and sustainable consumption. The goal is to provide a modern, intuitive, and secure environment where users can give a second life to tech and other items.

## üöÄ Key Features
This project is built to offer a complete e-commerce experience:

* **User Management:** Secure account creation and login system.
* **Product Listings:** Users can publish items with images, descriptions, prices, and categories.
* **Shopping System:** "Add to Cart" functionality for purchasing multiple items.
* **Wishlist:** Save favorite items for later.
* **Search & Filters:** Filter products by category, price, and condition.
* **User Dashboard:** A personal page to manage listed products.

## ‚öôÔ∏è How to Deploy (Local Server)

To run this project locally, you will need a local server environment like **XAMPP** or **Wampserver**.

1.  **Download and Install:**
    *   [Download XAMPP](https://www.apachefriends.org/) or [Download Wampserver](https://www.wampserver.com/).
2.  **Project Location:**
    *   Copy the project folder to the server's root directory:
        *   **XAMPP:** `C:\xampp\htdocs\secondavenue`
        *   **Wampserver:** `C:\wamp64\www\secondavenue`
3.  **Database Setup:**
    *   Start **Apache** and **MySQL** from your server's control panel.
    *   Open your browser and visit: `http://localhost/secondavenue/setup_db.php`
    *   Wait for the "success" message. This will automatically create the database and tables.
4.  **Access the Website:**
    *   Enter `http://localhost/secondavenue/website/index.php` in your browser.
5.  **Important Note:**
    *   Ensure your MySQL user is `root` and has no password (default for both XAMPP and WAMP). If you have a different configuration, update the credentials in `backend/db.php`.
